Class: GfxLayer

C64Style.GfxLayer

new GfxLayer(screenContext, canvas, props)

Graphics layer.
Extends C64Style.Layer
Generally, the use of C64Style.Screen.createLayer("GfxLayer") is preferred over creating layer by hand.
Parameters:
Name Type Description
screenContext C64Style.Screen The parent screen for this layer.
canvas CanvasContext The CanvasContext that this layer will draw to.
props Object The properties to create this layer with.
From C64Style.Layer:
  • width - number - The width of the layer. Should match Screen.
  • height - number - The height of the layer. Should match Screen.
GfxLayer doesn't require/support any properties beyond those used by C64Style.Layer.

render(time, diff)

Render the dirty elements on this layer. Calls clear for all dirty elements first, then calls render on each. Time and diff parameters are not directly used, they are made available for extension purposes, and passed on to clear and render for the same.

update(time, diff)

Update the layer. Calls update on each element. Checks for elements colliding with the layer boundary and emits events accordingly (event emitted from the elements themselves):
  • C64Style.EventType.ELEMENT_HIT_LEFT_EDGE
  • C64Style.EventType.ELEMENT_HIT_RIGHT_EDGE
  • C64Style.EventType.ELEMENT_HIT_TOP_EDGE
  • C64Style.EventType.ELEMENT_HIT_BOTTOM_EDGE
The data for these events consists of:
  • element : The element that hit the border
  • layer : This layer
Parameters:
Name Type Description
time number The current time (milliseconds)
diff number The difference between the last time and the current time (milliseconds)
