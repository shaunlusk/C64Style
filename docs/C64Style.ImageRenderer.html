renderImage(context, image, sx, sy, sWidth, sHeight, x, y, width, height, imageScaleX, imageScaleY)

Draws an image or portion of an image to the canvas.
Parameters:
Name Type Description
context CanvasContext The canvas to draw to.
image Image Image reference, created by <img> or new Image();
sx number The x starting point of a subsection of the image to draw.
sy number The y starting point of a subsection of the image to draw.
sWidth number The width of a subsection of the image to draw.
sHeight number The height of a subsection of the image to draw.
x number The target x position of the canvas to draw the image to.
y number The target y position of the canvas to draw the image to.
width number The target width of the drawn image. If different than the dimensions of the image subsection, the image subsection will be stretched or shrunk.
height number The target height of the drawn image. If different than the dimensions of the image subsection, the image subsection will be stretched or shrunk.
imageScaleX integer The amount to scale the drawn image horizontally.
imageScaleY integer The amount to scale the drawn image vertically.
