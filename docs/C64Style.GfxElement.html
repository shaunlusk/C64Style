Class: GfxElement

C64Style.GfxElement

new GfxElement(screenContext, parentLayer, props)

Graphics element base class.
Parameters:
Name Type Description
screenContext C64Style.Screen The target screen.
parentLayer C64Style.GfxLayer The parent layer that will draw this element.
props Object Properties for this GfxElement. Supports:
  • scaleX - integer - Horizontal scale of this element. Independent of screen scale.
  • scaleY - integer - Vertical scale of this element. Independent of screen scale.
  • hidden - boolean - Whether to hide this element.
  • x - number - The X coordinate for this element.
  • y - number - The Y coordinate for this element.
  • zIndex - number - The z-index; elements with higher zIndex values will be drawn later than those with lower values (drawn on top of those with lower values).

setMoveRates(xMoveRate, yMoveRate)

Set the horizontal and vertical movement rates for this element. Rates will be treated as approximately pixels per second. Negative values will move the element left for xMoveRate or up for yMoveRate. Zero values will halt movement on a given axis. Positive values will move the element right for xMoveRate or down for yMoveRate. Note that moveTo instructions will supercede movement rates in determining the element's position.
Parameters:
Name Type Description
xMoveRate number Horizontal movement rate
yMoveRate number Vertical movement rate
