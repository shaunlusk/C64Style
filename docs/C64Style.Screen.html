Class: Screen

C64Style.Screen

new Screen(targetDiv, config)

The Screen is the overriding container for all C64Style components. The Screen orchestrates updating and rendering its layers, propagates mouse events down to the layers, and notifies event listeners when events occur.
Parameters:
Name Type Description
targetDiv HTMLElement The target HTMLElement into which the screen and its layers will be built.
config Object Supported configuration properties:
  • scaleX - integer - the horizontal scale of the screen. Default: 1
  • scaleY - integer - the horizontal scale of the screen. Default: 1
  • cols - integer - The number of columns for the screen. Width will be sized accordingly: cols * C64Style.CELLWIDTH. Default: 40
  • rows - integer - The number of rows for the screen. Height will be sized accordingly: rows * C64Style.CELLHEIGHT. Default: 25
  • fpsElem - HTMLElement - Element to write Frames-per-second information to.
  • backgroundColor - Color - The color to set the screen background to. Default: C64Style.Color.BLUE
  • borderColor - Color - The color to set the screen border to. Default: C64Style.Color.LIGHTBLUE
  • borderSize - integer - The borderSize of the screen, in pixels. Default: 20

addLayer(layer)

Add a new C64Style.Layer to this screen. The preferred method of adding layers is via the createLayer() method, but this will also work. Layers will be rendered in FIFO order, so layers added later will be drawn on top of layers added earlier.

Name Type Description
layer C64Style.Layer The layer to add to the screen.
