Class: Layer

C64Style.Layer

new Layer(screenContext, canvas, props)

Abstract class for graphical layers on a C64Style.Screen.
Existing implementations: C64Style.TextLayer, C64Style.GfxLayer
Parameters:
Name Type Description
screenContext C64Style.Screen The parent screen
canvas CanvasContext The canvas context that this layer will draw to.
props Object The properties for this layer:
  • width - number - The width of the layer. Should match Screen.
  • height - number - The height of the layer. Should match Screen.
