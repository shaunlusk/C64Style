Class: TextLayer

TextLayer(props)

new TextLayer(props)

Text-only layer.
Extends Layer Generally, the use of C64Screen.createLayer("TextLayer") is preferred over creating layer by hand.
Parameters:
Name Type Description
props Object Properties
Properties
Name Type Attributes Default Description
screenContext Screen The parent screen.
canvasContextWrapper CanvasContextWrapper The canvasContextWrapper. This layer will draw to the canvas' context, via wrapper's exposed methods.
width number The width of the layer. Should match Screen.
height number The height of the layer. Should match Screen.
characterRenderer CharacterRenderer <optional>
 new CharacterRenderer The renderer to use to draw text. This can be shared with a renderer for drawing text elements. If a renderer is not provided, This TextLayer will create a CharacterRenderer.
textPrompt TextPrompt <optional>
 new TextPrompt A text prompt for this layer. If not provided, a TextPrompt will be created.
registerKeyHandler function <optional>
 window.document.addEventListener A function to register a key handler with; this allows keystrokes to be sent to the default TextPrompt. If not provided, window.document.addEventListener will be used.
scaleX integer <optional>
 1 The horizontal scale.
scaleY integer <optional>
 1 The vertical scale.
