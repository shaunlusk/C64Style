prompt(prompt, x, y, color, maxLength, callback)

Present a text prompt on the parent layer.
Parameters:
Name Type Description
prompt string The text to prompt the user with, e.g. "Enter your name:"
x integer The column to start rendering the text prompt to.
y integer The row to start rendering the text prompt to.
color Color The color for the text prompt.
maxLength integer The maximum allowed length for the input.
callback function The function to call when the user has hit the Enter key. The argument to the callback is the text the user entered.
