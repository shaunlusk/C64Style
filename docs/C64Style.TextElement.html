Class: TextElement

C64Style.TextElement

new TextElement(screenContext, parentLayer, props)

Element that draws a text string or a text symbol to a canvas.
Extends: C64Style.GfxElement
Parameters:
Name Type Description
screenContext C64Style.C64Screen The target screen.
parentLayer C64Style.GfxLayer The parent layer that will draw this element.
props Object Properties for this TextElement. Supports: From C64Style.GfxElement
  • scaleX - integer - Horizontal scale of this element. Independent of screen scale.
  • scaleY - integer - Vertical scale of this element. Independent of screen scale.
  • hidden - boolean - Whether to hide this element.
  • x - number - The X coordinate for this element.
  • y - number - The Y coordinate for this element.
  • zIndex - number - The z-index; elements with higher zIndex values will be drawn later than those with lower values (drawn on top of those with lower values).
For TextElement:
  • text - string - The text for this element. A text string or a symbol name is required for TextElement.
  • symbolName - string - The symbolName for this element. A text string or a symbol name is required for TextElement. Refer to symbol names in C64Style.CharacterMap.
  • color - Color - The color for the text. Default: C64Style.Color.LIGHTBLUE
  • backgroundColor - Color - The backgroundColor for the text. Default: no background color
  • characterRenderer - CharacterRenderer - The renderer to use to draw text. This can be shared with a renderer for drawing text elements. If a renderer is not provided, This TextLayer will create a C64Style.CharacterRenderer.
