Class: TextLayer

C64Style.TextLayer

new TextLayer(screenContext, canvas, props)

Text-only layer.
Extends C64Style.Layer
Generally, the use of C64Style.Screen.createLayer("TextLayer") is preferred over creating layer by hand.
Parameters:
Name Type Description
screenContext C64Style.Screen The parent screen for this layer.
canvas CanvasContext The CanvasContext that this layer will draw to.
props Object The properties to create this layer with.
From C64Style.Layer:
  • width - number - The width of the layer. Should match Screen.
  • height - number - The height of the layer. Should match Screen.
For TextLayer:
  • characterRenderer - CharacterRenderer - The renderer to use to draw text. This can be shared with a renderer for drawing text elements. If a renderer is not provided, This TextLayer will create a C64Style.CharacterRenderer.
  • textPrompt - TextPrompt - A text prompt for this layer. If not provided, a C64Style.TextPrompt will be created.
