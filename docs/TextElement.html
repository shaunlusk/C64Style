 -1 The z-index; elements with higher zIndex values will be drawn later than those with lower values (drawn on top of those with lower values).

color Color <optional>
 Color.LIGHTBLUE The color for the text.
backgroundColor Color <optional>
 none The backgroundColor for the text.
characterRenderer CharacterRenderer <optional>
 new CharacterRenderer The renderer to use to draw text. This can be shared with a renderer for drawing text elements. If a renderer is not provided, This TextElement will create a CharacterRenderer.
